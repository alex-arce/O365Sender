# O365Sender
Office 365 Mail tester

#### Compilation

Clone the repository:
`git clone https://github.com/alex-arce/O365Sender`

Build the binary:
```
cd cmd
./build.sh
```

#### Usage
```
./O365Sender -config o365sender.conf --from="user@domain.com" --to="user@domain.com" --subject="TEST SUBJECT" --text="TEST BODY"
```
